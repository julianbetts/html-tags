List of Standard HTML Tags with Definitions

  • <html> - Defines the root element of an HTML document.
  • <head> - Contains metadata information about the HTML document, including the title, scripts, and stylesheets.
  • <title> - Defines the title of the HTML document, which is displayed in the browser's title bar.
  • <body> - Contains the main content of the HTML document.
  • <header> - Defines a header section at the top of a document or a section.
  • <nav> - Defines a section of navigation links.
  • <main> - Defines the main content area of a document.
  • <article> - Defines a self-contained piece of content, such as a blog post or news article.
  • <section> - Defines a section of related content.
  • <aside> - Defines a section of content that is related to the main content but not part of it.
  • <footer> - Defines a footer section at the bottom of a document or a section.
  • <h1> to <h6> - Defines headings of different levels, with <h1> being the most important and <h6> the least important.
  • <p> - Defines a paragraph of text.
  • <a> - Defines a hyperlink to another web page or resource.
  • <img> - Displays an image on a web page.
  • <ul> - Defines an unordered list.
  • <ol> - Defines an ordered list.
  • <li> - Defines a list item within a list.
  • <table> - Defines a table with rows and columns.
  • <tr> - Defines a row within a table.
  • <td> - Defines a cell within a table.
  • <form> - Defines a form for user input.
  • <input> - Defines an input field within a form.
  • <button> - Defines a clickable button.
  • <textarea> - Defines a multiline input field.
